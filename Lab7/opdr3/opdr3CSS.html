• Waarom heeft de volgende CSS regel: .onbekend {background‐color:red;} geen effect als je deze toepast op een <td> cel in een <tr> rij met class="alt", maar wel in een <tr> rij zonder deze 'alt' class?

    • Omdat er al een stijl bepaalt is voor de tr met class="alt", namelijk een groene achtergrond. Als je een rode achtergrond zou willen geven moet je in het klasse "onbekend" het attribut "!important" achter de "background‐color" plaatsen zodat ze niet met de "alt" klasse rekening houdt (zie CSS van Persoonlijke pagina).
